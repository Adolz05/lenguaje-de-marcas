# Lenguaje de Marcas
Proyectos enfocados en la estructuración de datos y estándares web.

## Proyecto: LM_P2
Sitio web estático ambientado en Star Wars desarrollado desde cero.
- **Estructura**: Uso estricto de HTML5 semántico (`<header>`, `<main>`, `<aside>`, `<section>`).
- **Validación**: Implementación de formularios con validación por expresiones regulares (RegEx) para códigos de identificación.
- **Estándares**: Código validado mediante las herramientas oficiales del W3C para HTML y CSS.
